InterseccaoEntreTodasAsLinhas.calculate_intersection_original()

Calculate the intersection between all lines

Iterates through all lines and calculate each line’s intersection with every other line (including those that have already been visited). This is the original implementation.

InterseccaoEntreTodasAsLinhas.calculate_intersection_original2()

Calculate the intersection between all lines

Iterates through all lines and calculate each line’s intersection with every other line (excluding those that have already been visited).

InterseccaoEntreTodasAsLinhas.calculate_intersection_ss()

Calculate the intersection between lines using the subdivision matrix

Iterates through all lines on the global linhas array and calculates each line’s possible intersection set. A set Set[int] is used for its guarenteed uniqueness (an entry can be added only once).

InterseccaoEntreTodasAsLinhas.use_aabb()

Use AABB as the collision detection algorithm

Setup the script to run using the AABB collision algorithm. Set the global algo_init, algo_draw and algo_calculate_intersection functions to the specialized, local, AABB-related functions.

InterseccaoEntreTodasAsLinhas.use_original()
